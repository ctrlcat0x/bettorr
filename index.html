About Bettorr

Bettorr is a modern, minimalistic web interface for searching and downloading PC game torrents from multiple trusted sources. It features a beautiful dark theme, glassmorphism, and a blazing fast search experience.

  • Aggregates torrents from DODI, FitGirl, KaosKrew, OnlineFix, and Xatab
  • Smart search with autocomplete and source filtering
  • Direct magnet links for instant downloads
  • Responsive, mobile-friendly design
  • Open source, built with love for the gaming community

This project is for educational purposes only. Please comply with your local laws regarding torrent downloads.
